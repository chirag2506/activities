Objects in JavaScript

  • An object in JavaScript is an entity having state and behavior (properties and method). There are 3 ways to create objects:

    • By object literals
    • By creating instance of Object directly (using new keyword)
    • By using an object constructor (using new keyword)
  • Getting values from objects
    • Dot operator
    • Square brackets
  • Objects are mutable

  • Getting the description of a specific property in an object
